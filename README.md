# Reproducibility Package: Resource Overlap Driving Cultural Fusion in the NETP

This repository contains the raw data, source code, and methodological workflows for the study of **Majiayao and Zongri cultures (8.0–4.0 ka BP)** on the Northeastern Tibetan Plateau.

[![DOI](https://github.com/du17356830607/Cultural-fusion-raw-date)


## 📂 Repository Structure

### 1. `/Data` (Raw Datasets)
- `Majiayao_Samples_Table S6.csv.csv`: Verified coordinates of 1,239 Majiayao sites.
- `Zongri_Samples_Table S7`: Verified coordinates of 57 Zongri sites.
- `MaxEnt_Input_Samples_Table S4`: Verified coordinates of 37 Microlith sites.
- `Isotope_Raw Data_Table S8.csv`: Raw $^{87}Sr/^{86}Sr$ ratios from ZBK human remains and environmental baseline samples .

### 2. `/Code` (Python Scripts)
- `Agriculture_Suitability.docx`: Scripts for positive/negative geospatial factor normalization and Unsupervised Random Forest model for determining factor weights.
- `Cultural_Diffusion_Path.docx`: ArcPy-based script for generating Least Cost Paths (LCP) and diffusion routes.
- `Cultural_Hearth_Detection.docx`: Local Maxima algorithm for identifying cultural core areas.

### 3. `/Workflows`
- `Methods_workflow.docx`: **Essential Reading.** A step-by-step manual detailing the formulas, soil reclassification scores, and MaxEnt parameters used in the study.
